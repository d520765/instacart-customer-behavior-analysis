# instacart-customer-behavior-analysis
Полный анализ поведения покупателей Instacart с использованием Python, SQL и Power BI: подготовка данных, EDA, анализ покупок, повторяемости, категорий и интерактивный дашборд

Основные цели проекта
- Провести загрузку и очистку данных Instacart  
- Построить витрину данных (факт-таблицу) для SQL и Power BI  
- Выполнить исследовательский анализ (EDA)  
- Проанализировать поведение пользователей  
- Изучить повторяемость покупок  
- Определить топовые категории и товары  
- Визуализировать результаты в Power BI  
- Сформулировать бизнес-рекомендации

Используемые инструменты

Python 
- pandas, numpy  
- matplotlib  
- очистка данных  
- объединение таблиц  

SQL
- агрегаты, фильтры, группировка  
- топ товаров  
- активность по времени и дням недели  
- анализ повторных покупок

Power BI
- интерактивный дашборд  
- измерения и калькуляции DAX  
- фильтры по товарам/категориям  
- визуализация поведения пользователей
